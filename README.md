#### 題目說明

```
一計算費用程式，
平日一到五收費：大人 268 元、小孩 120 元
晚上及例假日：大人 368 元、小孩 150 元 並加收 10% 服務費
現舉辦周年促銷活動：三人同行一人免費，折扣最低價，十人以上再打 95 折
```
